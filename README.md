# Impollutionize: Enhance Your Knowledge of Pollution
## What is it? ##
Our GUI based Python application is a memory game about facts on pollution.
### How to play
There are three diferent gamemodes: Water Pollution, Air Pollution, and Light Pollution. Clicking on each will bring you to a new game about facts on the chosen gamemode.
Match all the terms and definitions to win! If you select two boxes and they don't match, the two boxes will be hidden the next time you click a box. You get infinite tries to match the boxes!
Note: the order of the terms and defenitions presented on screen will be different every time you start a new game, so memorizing the positions of everything in the hopes of beating a new game instantly will not work.

## Our gour goal
Our purpose for this app is to help you gain a deep understanding of pollution (specifically: water pollution, air pollution, and light pollution). This is because we believe if you know a lot about pollution, you can help prevent it.

## How did we build & pull it off?
We built this application using Python. We leveraged the library PySide6, and Qt Designer, to implement the Graphical User Interface.
Qt Designer was used to design the layout of our program. Qt Designer will then compile the .ui file into a Python file where we can reference/use in our code.

## Where was Google/AI used?
Google/AI was used for researching the facts about pollution, acquiring documentation of libraries, and troubleshooting a few last minute bugs in our program.
